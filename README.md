# Machine Learning Projects

This repository contains hands-on Machine Learning projects covering fundamental to intermediate concepts.  
The focus is on **data preprocessing, model building, evaluation, and visualization** using Python.

## 🔧 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib & Seaborn
- Scikit-learn

## 📌 Topics Covered
- Supervised Learning
  - Linear Regression
  - Logistic Regression
  - Decision Trees
- Unsupervised Learning
  - K-Means Clustering
  - DBSCAN
  - Hierarchical Clustering
- Model Evaluation (Accuracy, R², Confusion Matrix)

## 🎯 Objective
To build a strong foundation in Machine Learning by implementing algorithms on real-world datasets.

## 🚀 How to Run
1. Clone the repository  
2. Open notebooks in Jupyter / Google Colab  
3. Run cells sequentially
